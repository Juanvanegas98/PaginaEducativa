Préstamo de Equipos de Herramientas Ofimáticas

El área de Herramientas Ofimáticas ofrece equipos con software básico necesario para tareas administrativas, como procesamiento de texto, hojas de cálculo, presentaciones y gestión de correo electrónico. Estos equipos son ideales para estudiantes y profesionales que necesitan herramientas para tareas ofimáticas cotidianas.

Equipos Disponibles para Préstamo

  • Computadoras con Suite Ofimática: Equipos con programas como Microsoft Office (Word, Excel, PowerPoint), Google Docs, entre otros.
  • Escáneres y Impresoras: Equipos para digitalizar documentos y realizar impresiones de alta calidad.
  • Tabletas gráficas: Herramientas para la creación de diagramas y gráficos de alta precisión.
  • Proyectores: Dispositivos para hacer presentaciones en reuniones o exposiciones.
