Préstamo de Equipos para Diseño Gráfico

El área de Diseño Gráfico ofrece equipos especializados para aquellos estudiantes y profesionales que necesiten herramientas de alta calidad para desarrollar proyectos visuales. Estos equipos cuentan con software y hardware adecuados para tareas de diseño, edición y creación gráfica.

Equipos Disponibles para Préstamo

  • Computadoras con software de diseño: Adobe Photoshop, Illustrator, InDesign, entre otros.
  • Tabletas gráficas: Wacom para ilustraciones y diseño digital.
  • Cámaras profesionales: Para sesiones fotográficas y contenido visual.
  • Impresoras 3D: Para la creación de prototipos tridimensionales.
